安卓系统架构简述

Android系统的架构分为四层，从高层到底层依次为：
1、应用程序层（app+System apps）
2、应用程序框架层（Java API Framework）
3、系统运行库和运行环境层(Libraries + android Runtime)
4、Linux核心层(HAL+ Linux Kernel)
